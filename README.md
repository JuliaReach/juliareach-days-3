# juliareach-days-3

Website for the third edition of JuliaReach days 3, 9th - 10th December 2021.

### Instructions to deploy locally

1. Clone the repo

2. Activate and instantiate the environment.

```julia
$ julia --project=.

julia> ] activate
```

4. Do
```julia
using PkgPage; serve()
```

4. The website will open in your browser.
